# Scss-utils

scss style utils

## Install

```bash
npm i @justichentai/scss-utils
```

## Usage

```scss
@import '@justichentai/scss-utils';  
@import '@justichentai/scss-utils/basic';  
@import '@justichentai/scss-utils/test';  
@import '@justichentai/scss-utils/component';

@mixin basic;
```
